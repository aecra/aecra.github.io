网站强制使用 HTTPS

监听 80 端口上的所有请求，将其重定向到 443 端口，之后每个服务只需要监听 443 端口上对应的域名即可。

server {
    listen 80 default_server;
    server_name _;
    return 301 https://$host$request_uri;
}
